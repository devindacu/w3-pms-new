## Component Interaction Flows

### Channel Dashboard - Configure Channel
```
1. User clicks "Channels" tab
2. User clicks "Configure" on Booking.com card
3. Dialog opens with configuration form
4. User enters:
   - API Key: *************
   - Property ID: 12345
   - Sync Interval: Every 5 minutes
   - Auto-sync: ON
5. User clicks "Test Connection"
6. Loading spinner appears
7. Success message: "âœ… Connection successful"
8. User clicks "Save"
9. Dialog closes
10. Channel card updates with new settings
11. Toast notification: "Configuration saved"
```

### Enhanced Dashboard - Drag Widget
```
1. User clicks "Edit Dashboard"
2. Edit mode activates
3. Drag handles (â‰¡â‰¡â‰¡) appear on widgets
4. User clicks and holds Revenue widget
5. Cursor changes to grabbing hand
6. User drags widget to right
7. Other widgets shift to make space
8. User releases mouse
9. Widget drops into new position
10. Layout updates
11. User clicks "Done"
12. Configuration saved to KV storage
13. Toast: "Dashboard layout saved"
```

### Floor Plan - Book Room
```
1. User navigates to Floor Plan
2. User selects "Floor 1"
3. Green (available) rooms display
4. User clicks Room 101
5. Room details popup opens showing:
   - Room number, type, price
   - Features list
   - "Book Now" button
6. User clicks "Book Now"
7. Booking dialog opens (or redirects to reservations)
8. User completes booking form
9. Booking saved
10. Room status changes to Reserved (blue)
11. Floor plan updates in real-time
```
